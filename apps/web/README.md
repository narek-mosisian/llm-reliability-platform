# LLM Reliability Web

Frontend web application for the LLM Reliability Platform.

This service will provide the public landing page, local demo UI, evidence pages and project documentation views.

## Current scope

This initial frontend skeleton includes:

- Next.js App Router setup
- TypeScript configuration
- ESLint configuration
- Basic root layout
- Initial landing/demo placeholder page
- Basic global styling

## Local development

Install dependencies:

```bash
pnpm install
````

Run the development server:

```bash
pnpm dev
```

Run linting:

```bash
pnpm lint
```

Run the production build:

```bash
pnpm build
```

Open the app locally:

```text
http://localhost:3000
```
